最简单无脑的手册:  
1. git config --global user.name "__name__"
2. git config --global user.email "email@url.com"
3. git clone __url__ (复制项目,本地写程序,修改代码后)
4. git add __name__.cpp 
5. git commet -m'更新说明'
6. git push
	1. 如果是Github上的远端仓库被改动了,就不行了,得先 git pull
	2. 然后 git push
7. 等它完事儿
